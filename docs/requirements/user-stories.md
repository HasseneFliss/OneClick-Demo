# User Stories

## Stories

### US-001: User Registration

**Priority:** Medium
**Story Points:** TBD

**Description:**
As a new user, I want to register with email and password so I can access the application

**Acceptance Criteria:**
- [ ] User can enter email and password
- [ ] Password must be at least 8 characters
- [ ] User receives confirmation email

### US-002: User Login

**Priority:** Medium
**Story Points:** TBD

**Description:**
As a registered user, I want to login with my credentials so I can access my account

**Acceptance Criteria:**
- [ ] User can login with email/password
- [ ] Invalid credentials show error
- [ ] Session is maintained

### US-003: Dashboard View

**Priority:** Medium
**Story Points:** TBD

**Description:**
As a logged-in user, I want to see a dashboard with my key metrics

**Acceptance Criteria:**
- [ ] Dashboard shows user info
- [ ] Dashboard displays activity summary
- [ ] Data refreshes on page load
